# 🎮 Instagram Reach Simulator (AI-Powered)

An interactive **machine learning-powered web app** that predicts the reach of an Instagram post based on engagement metrics like likes, comments, shares, and more.

This project transforms data science into a **gamified experience**, allowing users to experiment with different strategies and understand how content performance impacts reach.

---

## 🚀 Features

* 🎮 **Gamified UI** – Adjust sliders like a game to simulate engagement
* 🤖 **Machine Learning Model** – Predicts Instagram reach using regression
* 📊 **Real-time Charts** – Visualize engagement breakdown instantly
* 🧠 **AI Insights** – Get suggestions to improve reach
* 🎯 **Challenge Mode** – Set target reach and test your strategy
* 📈 **Performance Levels** – Beginner → Growing Creator → Viral Creator

---

## 🧠 How It Works

1. User inputs engagement metrics:

   * Likes 👍
   * Saves 💾
   * Comments 💬
   * Shares 🔁
   * Profile Visits 👤
   * Follows ➕

2. The trained ML model processes inputs

3. The app predicts:

   * 📊 Expected Reach
   * 🎯 Performance Level
   * 🧠 Strategy Insights

---

## 🛠️ Tech Stack

* **Frontend/UI:** Streamlit
* **Machine Learning:** Scikit-learn (PassiveAggressiveRegressor)
* **Data Processing:** Pandas, NumPy
* **Model Persistence:** Joblib

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/instagram-reach-simulator.git
cd instagram-reach-simulator
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Train the model (optional)

```bash
python train_model.py
```

### 4️⃣ Run the app

```bash
streamlit run app.py
```
